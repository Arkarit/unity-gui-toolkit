## Recommended Image Sizes
- Screenshots: 1280x720 or 1920x1080 (PNG format)
- Diagrams: SVG preferred (scalable), or PNG at 2x resolution
- Inspector windows: Crop to relevant area, add border if helpful
- Workflows: Use tools like draw.io, Lucidchart, or Figma

## Style Guidelines
- Use consistent colors (match Unity's dark theme if possible)
- Add arrows and annotations where helpful
- Highlight important UI elements with colored boxes
- Use readable fonts (12pt minimum)
- Keep diagrams simple and focused
