Detailed Description

static class for determining loca plural forms.

This class consists of an internal part (this one) and a user part, which is generated by LocaPluralProcessor according to the language rules in the PO files. The generated class implements GetPluralIdx. The "partial" feature of C# feels a bit weird and awkward, but it is very handy - it's the "poor man's inheritance" for static classes.
