Detailed Description

Basic UI class.

Basic UI class which defines virtual methods for the most common Unity functions Awake(), OnEnable(), OnDisable() and OnDestroy().

Attention
Be sure to always call the base class methods if you inherit from UiThing

Additionally, it offers event handling plus some convenience functions.

◆ InitEvents()

void GuiToolkit.UiThing.InitEvents ( )

Install Event handlers on disabled objects.

Unity unfortunately has NO reliable "OnCreate" callback:
Constructors should not be used according to Unity, and Awake() is only called on active game objects.
So if you e.g. want to install event handlers on components, which are disabled on creation (e.g. to enable them on event), you're lost.
In such cases, you have to call InitEvents() manually.
