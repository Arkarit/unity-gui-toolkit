Detailed Description

General Editor Utility.

This is a collection of common editor helper functions.

Note: This file must reside outside of an "Editor" folder, since it must be accessible from mixed game/editor classes (even though all accesses are in #if UNITY_EDITOR clauses) See https://answers.unity.com/questions/426184/acces-script-in-the-editor-folder.html for reasons.
