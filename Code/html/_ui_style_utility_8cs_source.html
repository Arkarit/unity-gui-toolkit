40 // Find an example in UiDistortGroup, which sets a distort modifier component enabled or disabled according to its needs.
41 // As the complete distort modifier might be disabled by the skin/style, this might interfere.
42 // Thus, UiDistortGroup calls ReApplyAppliers() after handling its changes to ensure the component is not enabled if forbidden by style.
43 // This is better than reapplying the complete skin, but not completely lightweight, since components need to be collected, so handle with care.
