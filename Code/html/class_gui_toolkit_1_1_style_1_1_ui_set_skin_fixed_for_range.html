Detailed Description

This component is intended to suppress skin changes in a range of UiAbstractApplyStyleBase instances. It can be handy if you want varying skins in one screen, but a fixed skin in another.

In case of nested UISetSkinFixedForRange, the outer one does not collect the children of the inner one.
