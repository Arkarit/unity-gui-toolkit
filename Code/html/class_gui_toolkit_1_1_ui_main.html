◆ IsAwake

bool GuiToolkit.UiMain.IsAwake
staticget

This getter is especially important for bootstrapping. If you have a separate routine for setting up your application, there might be some state, where UiMain has been already loaded, but isn't functional yet (UiMain.Awake() not yet called) In that case, loop in a coroutine until IsAwake is true, before you use it.
