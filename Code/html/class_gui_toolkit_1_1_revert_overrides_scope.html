Detailed Description

When writing editor code which modify game objects and components in the currently edited scene, it often comes to unwanted property overrides of prefabs. This scope class can avoid this.

Consider in an editor code you want to render the currently selected camera in isometric mode into a temporary render texture. Simple to do: create a temporary render texture and assign it to the camera, change camera projection, render, and afterwards restore the previous camera settings / render texture. Now if the camera is part of a prefab instance, you have created some unwanted overrides, especially the render texture and the camera projection.

RevertOverridesScope helps to avoid this; it saves all property modifications for one or more specific objects at beginning of scope and restores it at end of scope, omitting all changes in between.
