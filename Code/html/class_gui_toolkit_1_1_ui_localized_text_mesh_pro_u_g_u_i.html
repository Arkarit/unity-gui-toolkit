Detailed Description

TextMeshProUGUI subclass with automatic localization support. When AutoLocalize is enabled, setting the LocaKey property immediately applies the translated text. Re-translates on language changes. Replaces the deprecated UiAutoLocalize component.

◆ text

override string? GuiToolkit.UiLocalizedTextMeshProUGUI.text
getset

Overrides the base TextMeshProUGUI.text property to intercept external writes. When AutoLocalize is enabled:

  • Writes from outside this class are treated as new LocaKey assignments and trigger translation.
  • A warning is logged in the editor if a key is already set (suggests using LocaKey instead). When disabled, behaves identically to the base property.

The re-entry guard (m_isSettingInternally) is required because ApplyTranslation writes through base.text, which routes back into this same override. Without the guard that call would be misidentified as an external write and loop indefinitely.
