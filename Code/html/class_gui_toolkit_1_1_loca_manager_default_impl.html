Translates a pluralized string key according to the active language's plural rules. Uses LocaPlurals.GetPluralIdx to determine which form to select based on _n . If plural index is 0, delegates to the singular Translate(string, string, RetValIfNotFound) overload. In "dev" mode, returns the plural key unchanged. Otherwise looks up the plural forms in the translation dictionary.
