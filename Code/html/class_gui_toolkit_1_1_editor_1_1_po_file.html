Detailed Description

Represents a parsed GNU gettext PO or POT file. Provides parsing from raw content, serialization back to PO format, and lookup building. String values stored in entries are unescaped (actual characters); the header msgstr is kept in its raw escaped form to preserve continuation-line structure during round-trips.
