Detailed Description

Generic base class with gettext-style localization helper methods.

Similar to LocaMonoBehaviour but for non-MonoBehaviour classes. Provides convenience wrappers around LocaManager using standard gettext naming conventions.

Note: The "_" shortcut is unusual in C# but standard in gettext/PO/POT tools.
